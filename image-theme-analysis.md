# 명언사주 기존 이미지 색상 테마 분석

## 핵심 색상 팔레트
- **배경**: 크림/베이지 (#f5f0e8, #ede5d5) — 한지(韓紙) 느낌의 따뜻한 오프화이트
- **먹/잉크**: 짙은 차콜 (#2a2520, #1a1510) — 수묵화 느낌
- **골드 포인트**: 황금 (#c9a84c, #b8860b) — 원형 붓터치, 테두리
- **강조 빨강**: 인장/낙관 (#c0392b, #8b1a1a) — 명언사주 도장
- **산수화 회색**: 연한 회색 (#8a8a8a, #b0a898) — 산, 구름
- **매화/꽃**: 붉은 분홍 (#d4606a) — 매화 꽃잎
- **대나무**: 짙은 녹색 (#2d4a2d)

## 디자인 무드
- **스타일**: 한국 전통 수묵화 (수묵담채화) + 현대 프리미엄
- **배경**: 밝은 한지 베이지 (어두운 네이비 아님!)
- **텍스처**: 종이 질감, 붓터치, 수묵 번짐
- **레이아웃**: 원형 금빛 테두리 카드, 매화/대나무 장식
- **로고**: 붉은 인장(낙관) 스타일 — 명언사주 한자

## 섹션별 이미지 특징
- **section1-hero**: 어두운 배경 (먹빛 차콜) + 한복 입은 여성 + 골드 원형 엠블럼
- **section2-problems**: 밝은 한지 베이지 + 매화나무 + 4개 빈 카드 프레임 (골드 테두리)
- **section3-system**: 밝은 한지 + 매화/대나무 + 중앙 금빛 원 + 11개 원형 슬롯
- **section4-report-mockup**: 밝은 베이지 배경 + 2권의 책 (흰색/먹색 표지, 금빛 원)
- **section7-pricing**: 밝은 한지 + 매화/대나무 + 2개 카드 프레임 (01/02 번호)

## 결론
명언사주 브랜드는 **밝은 한지 베이지 배경 + 수묵화 + 골드 + 붉은 인장** 조합.
참고 사이트(월백당)의 어두운 네이비와 다름. 밝고 전통적인 서예/한지 미학.
